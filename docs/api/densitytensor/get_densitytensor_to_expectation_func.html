get_densitytensor_to_expectation_func

qujax.get_densitytensor_to_expectation_func(hermitian_seq_seq, qubits_seq_seq, coefficients)

Takes strings (or arrays) representing Hermitian matrices, along with qubit indices and a list of coefficients and returns a function that converts a densitytensor into an expected value.

Parameters:
  • hermitian_seq_seq (Sequence[Sequence[Union[str, jnp.ndarray]]]) – Sequence of sequences of Hermitian matrices/tensors. Each Hermitian matrix is either represented by a tensor (jnp.ndarray) or by a list of ‘X’, ‘Y’ or ‘Z’ characters corresponding to the standard Pauli matrices. E.g. [[‘Z’, ‘Z’], [‘X’]]
  • qubits_seq_seq (Sequence[Sequence[int]]) – Sequence of sequences of integer qubit indices. E.g. [[0,1], [2]]
  • coefficients (Union[Sequence[float], jnp.ndarray]) – Sequence of float coefficients to scale the expected values.
Returns:

Function that takes densitytensor and returns expected value (float).
Return type:

Callable[[jnp.ndarray], float]
